# Phase Template

Use this template for any new bounded OpenCrux phase or slice document that deserves its own artifact.

## Title

Short phase name.

## Goal

- one clear statement of what this slice is meant to achieve

## Scope

In scope:

- bounded work items only

Out of scope:

- explicit exclusions to prevent drift

## Discovery

- current context
- relevant artifacts read first
- assumptions or constraints

## Implementation

- exact surfaces expected to change
- files or systems intentionally left untouched

## Verification

- automated checks to run
- manual checks if relevant
- promotion or rejection criteria

## Review

- residual risk
- open questions
- decision summary

## Next Step

- exact recommended next slice
- exact first command or artifact to inspect next session
